# 01 — 需求规格

## 项目名称

我的日记（Diary PWA）

## 用户画像

个人用户，iPhone 使用者，希望用简单的 App 记录日常生活。

## 核心功能

| 编号 | 功能 | 说明 |
|------|------|------|
| F1 | 创建日记 | 输入标题 + 正文，保存到本地 |
| F2 | 编辑日记 | 点击已创建的日记进入编辑模式，修改后保存 |
| F3 | 删除日记 | 长按或滑动删除日记，带确认提示 |
| F4 | 插入图片 | 从相册选择照片 或 直接拍照，自动压缩 |
| F5 | 插入视频 | 从相册选择视频 或 直接录制 |
| F6 | 日记列表 | 按日期倒序显示，卡片形式，按日期分组 |
| F7 | 全文搜索 | 搜索标题和正文内容，实时过滤结果 |
| F8 | 日历视图 | 月历形式，有日记的日期显示标记，点击查看 |
| F9 | 离线访问 | 无网络时也能打开 App 浏览已有日记 |

## 非功能需求

- 适配 iOS 15 及以上版本 Safari
- 支持 PWA "添加到主屏幕"，全屏运行
- 界面响应式，适配 iPhone 各尺寸
- 数据仅存储在设备本地，不上传

## 不做的功能（V1）

- 云端同步
- 密码/面容锁定
- 日记导出
- 心情/天气标签
- 位置记录
- 多语言（仅简体中文）
